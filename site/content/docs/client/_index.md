Ways to interact with Composia:

- **Web UI** — browser-based dashboard for managing services, nodes, tasks, backups, and settings. Open `http://localhost:3000` after starting the stack.
- **CLI** — `composia` command-line tool for scripting and terminal workflows. Install via [package managers](/docs/installation/package-managers/) or download from the [releases page](https://forgejo.alexma.top/alexma233/composia/releases).

Both the web UI and CLI talk to the same [controller API](/docs/developer-guide/api/).
