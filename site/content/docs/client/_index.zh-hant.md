與 Composia 互動的方式：

- **Web UI** — 基於瀏覽器的儀表板，用於管理服務、節點、任務、備份與設定。啟動堆疊後開啟 `http://localhost:3000`。
- **CLI** — `composia` 命令列工具，用於腳本編寫與終端機工作流程。可透過[套件管理器](/docs/installation/package-managers/)安裝，或從[發行版本頁面](https://forgejo.alexma.top/alexma233/composia/releases)下載。

Web UI 與 CLI 都與相同的[控制器 API](/docs/developer-guide/api/) 溝通。
