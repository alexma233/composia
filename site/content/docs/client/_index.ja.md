Composia と対話する方法:

- **Web UI** — サービス、ノード、タスク、バックアップ、設定を管理するブラウザベースのダッシュボード。スタック起動後に `http://localhost:3000` を開きます。
- **CLI** — スクリプトとターミナルワークフローのための `composia` コマンドラインツール。[パッケージマネージャー](/docs/installation/package-managers/) 経由でインストールするか、[リリースページ](https://forgejo.alexma.top/alexma233/composia/releases) からダウンロードします。

Web UI と CLI はどちらも同じ [コントローラー API](/docs/developer-guide/api/) と通信します。
